2 Approche Anova

On considère ici le temps comme une variable qualitative à

length(levels(data2$Day))
## [1] 21

modalités. On se base sur le tutoriel https://www.datanovia.com/en/fr/lessons/anova-sur-mesures-repetees-dans-r/#anova-a-deux-facteurs-sur-mesures-repetees pour l’analyse. Commençons par étudier la normalité

shap <- data2 |>
  group_by(Day, groupe) |>
  shapiro_test(Poids)
datatable(shap)
shap |> filter(p<0.05)
## # A tibble: 4 × 5
##   groupe Day   variable statistic      p
##   <fct>  <fct> <chr>        <dbl>  <dbl>
## 1 CaCTL  13    Poids        0.825 0.0395
## 2 CaCTL  16    Poids        0.792 0.0166
## 3 CaCTL  17    Poids        0.822 0.0359
## 4 CaCTL  9     Poids        0.824 0.0380

L’hypothèse de normalité est (quasiment) tout le temps accepté.

Les répétitions portant sur le temps (et non sur les groupes), on fait l’ANOVA à deux facteurs avec (voir https://www.math.univ-toulouse.fr/~besse/Wikistat/pdf/st-m-modmixt7-mesrepet.pdf)

  • groupe comme effet inter
  • Day comme effet intra
res.aov <- anova_test(
  data = data2, dv = Poids, wid = Id,
  between = groupe,
  within = Day
)
get_anova_table(res.aov)
## ANOVA Table (type III tests)
## 
##       Effect  DFn    DFd     F        p p<.05   ges
## 1     groupe 2.00  28.00 0.375 6.91e-01       0.024

## 3 groupe:Day 8.46 118.46 4.637 4.27e-05     * 0.031

L’interaction est significative.

On peut aller un peu plus loin avec les tests post hoc.

2.1 Effet du groupe

On étudie ici l’effet groupe à chaque pas de temps :

one.way <- data2 %>% 
  group_by(Day) %>%
  anova_test(dv = Poids, wid = Id, between = groupe) %>%
  get_anova_table() %>%
  adjust_pvalue(method = "bonferroni")
one.way %>% as_tibble() %>% 
  mutate(Day=as.numeric(as.character(Day))) %>% 
  arrange(Day) %>% datatable()

3 Modèle mixte

On revient ici sur les modèles mixtes, en considérant la variable temps comme continue (ce qui semble faire du sens vu qu’on a de nombreux pas de temps).

On considère l’interaction temps:groupe et on compare les modèles avec :

  • juste la constante comme effet aléatoire
  • la constante et le temps
m1 <- lme(Poids~1+groupe+Day+Day:groupe,
         random=list(Id=pdDiag(~1)),
         data=data21)
